# Penyimpanan Data

Data website dipisah dari file tampilan.

- Data murid, absen, progress, kode parent: data/les-koding.db
- File upload PPT, cover, sertifikat: uploads/
- File tampilan: public/

Mengubah style.css, parent.html, admin.html, atau app.js tidak menghapus data.
Data hilang hanya kalau folder data/ atau uploads/ terhapus, atau hosting gratis mereset storage.
